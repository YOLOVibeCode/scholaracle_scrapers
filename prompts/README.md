# AI Prompt Templates

These prompt templates let you use your preferred AI tool (ChatGPT, Claude, Cursor, etc.)
to create and troubleshoot scrapers.

## How to Use

1. Open the prompt file you need (e.g., `generate-scraper.md`)
2. Fill in the bracketed placeholders with your details
3. Copy the entire prompt into your AI tool
4. For best results, also paste `context/scraper-spec-context.md` as a separate context document

## Available Prompts

| File | Purpose |
|------|---------|
| `generate-scraper.md` | Create a new scraper for any educational platform |
| `troubleshoot-scraper.md` | Diagnose and fix scraper errors |
| `customize-scraper.md` | Add features or modify an existing scraper |
| `context/scraper-spec-context.md` | Full specification context — paste this alongside any prompt |
